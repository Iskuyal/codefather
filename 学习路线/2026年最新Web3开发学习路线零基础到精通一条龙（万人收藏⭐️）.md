## 开篇介绍

Web3 是基于区块链技术构建的去中心化网络，核心理念是将数据和资产的所有权还给用户。从 DeFi 到 NFT、从 DAO 到 GameFi，Web3 正在重塑互联网格局。

**为什么要学 Web3 开发？**

Web3 是当前最具创新性的技术方向之一。掌握 Web3 开发，不仅能让你参与 “区块链革命”，还能获得可观的薪资回报。一线城市的 Web3 工程师平均薪资在 30-60K。而且 Web3 开发工程师的需求量持续增长，职业发展空间广阔。

![](https://pic.yupi.icu/1/image-20251202192055339.png)

不过鱼皮友情提示：Web3 领域存在钓鱼网站、私钥泄露等安全风险，开发者要特别注重安全，保护用户的私钥和资产。而且很多公司可能会通过高薪来钓鱼，甚至通过让你拉取个开源代码来盗取你的加密货币，谨防受骗！

本篇学习路线侧重于 Web3 前端开发，包括使用 ethers.js/web3.js 与区块链交互、集成钱包（MetaMask）、处理交易、连接 IPFS 等。关于智能合约开发的详细学习，可以查看 [智能合约开发学习路线](https://www.codefather.cn/course/1789189862986850306/section/1990756156976443394)

前端开发的详细学习，可以查看 [前端开发学习路线](https://www.codefather.cn/course/1789189862986850306/section/1789190394078011393)。

4）记住，永远不要在主网上测试！可以使用 Sepolia 等测试网络进行开发和测试，避免资产损失。测试网的 ETH 可以通过水龙头免费领取。



## 阶段 1：Web3 基础（7-15 天，仅供参考）

### 学习目标

理解 Web3 的核心概念，掌握基本的区块链知识。



### 知识点

**Web3 基础概念【必学】：**

- Web3 的定义和特点
- Web3 和 Web2 的区别
- 去中心化应用（DApp）
- 区块链、钱包、Gas

**以太坊基础【必学】：**

- 以太坊账户
- 交易和区块
- 智能合约【简单了解】

**开发工具【必学】：**

- MetaMask 钱包
- Etherscan 区块链浏览器
- 测试网络（Sepolia）



### 学习建议

1）Web3 的核心是去中心化，用户拥有自己的数据和资产，理解 Web3 和 Web2 的本质区别非常重要。

![](https://pic.yupi.icu/1/mm-mobile-hero.png)

3）Etherscan 是以太坊的区块链浏览器，可以查看交易、区块、合约等信息。要学会使用 Etherscan 查询链上数据。

## 阶段 2：ethers.js / web3.js（15-25 天，仅供参考）


ethers.js 和 web3.js 是跟以太坊交互的 JavaScript 库，是开发 Web3 前端应用的核心工具。



### 学习目标

掌握 ethers.js，能够使用 JavaScript 和区块链交互。



### 知识点

**ethers.js 基础【必学，推荐】：**

- Provider（连接以太坊网络）
- Signer（签名交易）
- Contract（合约交互）
- 读取区块链数据

**web3.js【建议学】：**

- web3.js 的使用
- ethers.js 和 web3.js 的对比

**合约交互【必学】：**

- 调用合约的只读函数
- 调用合约的写函数（发送交易）
- 监听合约事件
- ABI 和合约地址



### 学习重点

1）ethers.js 和 web3.js 是两大以太坊 JavaScript 库。ethers.js 更轻量、更现代、API 更友好，建议优先学习 ethers.js。web3.js 是老牌库，功能更全面。

2）Provider 用于连接以太坊网络，Signer 用于签名交易。要理解 Provider 和 Signer 的区别和使用场景。

3）ABI（Application Binary Interface）是合约的接口定义，前端通过 ABI 和合约交互。要理解 ABI 的作用。

![](https://pic.yupi.icu/1/image-20251202192634040.png)

## 阶段 3：钱包集成和 DApp 开发（15-25 天，仅供参考）

### 学习目标

钱包集成是 DApp 开发的第一步，要学会连接 MetaMask、获取账户、签名交易等。

本阶段的目标是掌握钱包集成，能够开发完整的 DApp 前端。



### 知识点

**钱包集成【必学】：**

- MetaMask 连接
- 账户授权
- 网络切换
- 交易签名

**DApp 前端开发【必学】：**

- React / Vue + ethers.js
- 合约状态展示
- 交易发送
- 事件监听

**IPFS【建议学】：**

- 去中心化存储
- 上传和获取文件
- IPFS 网关

**Web3 UI 库【建议学】：**

- RainbowKit（钱包连接）
- wagmi（React Hooks）
- web3-react



### 学习建议

1）DApp 的前端开发和传统 Web 开发类似，主要区别是需要和区块链交互。可以使用 React 或 Vue，结合 ethers.js 开发 DApp。

2）IPFS 是去中心化存储方案，可以用于存储 DApp 的前端文件、NFT 的元数据等。理解 IPFS 的基本原理和使用方法。

![](https://pic.yupi.icu/1/image-20251202192758432.png)

3）几个非常重要的安全提醒：

   - 永远不要在前端代码中暴露私钥
   - 验证用户签名的交易参数，防止钓鱼攻击
   - 提供清晰的交易确认界面，让用户知道在签什么
   - 检测和警告用户钓鱼网站
   - 使用 HTTPS 保护网站安全



### 学习资源

- [如何使用 ethers.js 连接 MetaMask](https://learnblockchain.cn/article/11752)：详细教程



## 阶段 4：项目实战（20-40 天，仅供参考）

### 学习目标

通过实际项目巩固所学知识，积累 Web3 项目经验。



### 学习建议

1）从简单 DApp 开始：如展示账户余额、发送交易等，熟悉 Web3 开发流程。

2）开发完整 DApp：开发一个完整的 DApp（如 NFT 市场、去中心化投票、简单的 DeFi 应用），体验前后端开发。

3）部署 DApp：将 DApp 部署到 IPFS 或 Vercel，让用户可以访问。

4）关注用户体验：Web3 DApp 的用户体验相比 Web2 应用较差（如需要连接钱包、确认交易、等待区块确认等），要尽可能优化用户体验。



### 项目推荐

**入门级项目：**

- 简单的代币转账 DApp
- NFT 展示页面
- 去中心化投票
- 区块链浏览器（简化版）

**进阶级项目：**

- NFT 市场（Mint、交易）
- 去中心化交易所前端
- DeFi 借贷协议前端
- DAO 治理界面

## 尾声

Web3 是下一代互联网的愿景，代表了去中心化、用户主权的未来。Web3 开发让前端工程师也能参加区块链革命，开发去中心化应用。
