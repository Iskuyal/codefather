## 开篇介绍

HarmonyOS 鸿蒙系统是华为开发的面向全场景的分布式操作系统，于 2019 年首次发布。鸿蒙不仅是一个手机操作系统，更是一个万物互联的操作系统，可以运行在手机、平板、智能手表、智能家居、车机等各种设备上，实现多设备协同和分布式能力。

2024 年是鸿蒙原生应用的元年，华为宣布鸿蒙 NEXT（纯血鸿蒙）不再兼容 Android 应用，要求所有应用都要开发鸿蒙原生版本。这为鸿蒙开发者带来了巨大的机遇，鸿蒙应用开发成为了 2025 年最热门的技术方向之一。

![](https://pic.yupi.icu/1/interlace,1.jpeg)

鸿蒙应用开发使用 ArkTS 语言（基于 TypeScript 扩展）和 ArkUI 框架（声明式 UI 框架）。ArkTS 语法简洁，类似于 TypeScript；ArkUI 采用声明式编程，类似于 React 和 SwiftUI，学习曲线相对平缓。而且华为提供了 [DevEco Studio](https://developer.huawei.com/consumer/en/deveco-studio/) 作为官方 IDE，功能强大，开发体验很好。

![DevEco Studio](https://pic.yupi.icu/1/idepc.png)

**为什么要学鸿蒙开发？**

鸿蒙是国产操作系统的希望，国家大力支持鸿蒙生态的发展。随着华为手机和鸿蒙设备的普及，鸿蒙应用的需求量会越来越大。掌握鸿蒙开发，不仅能让你抓住新技术的红利期，还能为国产操作系统的发展贡献力量。而且鸿蒙开发者的薪资普遍较高，一线城市的鸿蒙开发工程师平均薪资在 20-40K。

![](https://pic.yupi.icu/1/image-20251127145200299.png)

在 AI 时代，鸿蒙也在积极拥抱 AI 技术。华为推出了盘古大模型、小艺助手等 AI 产品，鸿蒙系统内置了 AI 能力，开发者可以方便地调用 AI 接口。学习鸿蒙开发，结合 AI 技术，将为你打开新的职业空间。

## 整体学习建议

1）ArkTS 和 TypeScript 的关系：ArkTS 是基于 TypeScript 扩展的语言，语法基本相同。如果你会 TypeScript 或 JavaScript，学习 ArkTS 会非常快。

2）ArkUI 和其他声明式 UI 的关系：ArkUI 采用声明式编程，和 React、SwiftUI、Flutter 的设计理念类似。如果你会这些框架，学习 ArkUI 会更轻松。

3）重点学习鸿蒙特色：鸿蒙的特色是 **分布式能力和多设备协同**，要重点学习这些特性。

4）多看官方文档：华为提供了详细的官方文档和 [视频教程](https://developer.huawei.com/consumer/cn/codelabsPortal/serviceTypes)，质量很高。遇到问题时，优先查阅官方文档。

## 阶段 1：ArkTS 语言（5-15 天，仅供参考）


ArkTS 是鸿蒙应用开发的核心语言，基于 TypeScript 扩展而来，专为鸿蒙生态设计。



### 学习目标

掌握 ArkTS 编程语言，能够编写鸿蒙应用的脚本代码。

### 知识点

**ArkTS 基础语法【必学】：**

- 变量和常量（let、const）
- 数据类型（number、string、boolean、Array）
- 运算符和表达式
- 控制流（if、switch、for、while）
- 函数
- 装饰器（@Entry、@Component、@State 等）

**TypeScript 特性【必学】：**

- 类型注解
- 接口（interface）
- 类（class）
- 泛型
- 模块



### 学习建议

1）ArkTS 在 TypeScript 基础上增加了装饰器等特性，装饰器是 ArkUI 的核心，用于声明组件、状态等。

2）如果你会 TypeScript，可以快速过一遍 ArkTS，重点学习装饰器等鸿蒙特有的特性。

3）如果不会 TypeScript，建议先学习 TypeScript 基础。完整的 TypeScript 学习路线可以参考：[TypeScript 学习路线](https://www.codefather.cn/course/1789189862986850306/section/1990753941477961730)



### 学习资源

- ⭐ [ArkTS 官方文档](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides-V5/arkts-get-started-V5)：官方文档
- [TypeScript 学习路线](https://www.codefather.cn/course/1789189862986850306/section/1990753941477961730)：完整的 TypeScript 学习路线



## 阶段 2：ArkUI 基础（5-10 天，仅供参考）

### 学习目标

掌握 [ArkUI 框架](https://developer.huawei.com/consumer/cn/arkui/)，能够开发鸿蒙应用的界面。



### 知识点

**开发环境【必学】：**

- DevEco Studio 的安装和配置
- 项目的创建和运行
- 模拟器和真机调试

**基础组件【必学】：**

- Text、Image、Button
- Column、Row、Stack
- List、Grid
- Scroll

**布局【必学】：**

- 线性布局（Column、Row）
- 层叠布局（Stack）
- 弹性布局（Flex）
- 相对布局（RelativeContainer）

**状态管理【必学】：**

- @State：组件内状态
- @Prop：父子组件单向传递
- @Link：父子组件双向同步
- @Provide 和 @Consume：跨组件传递

**路由导航【必学】：**

- 页面路由（router）
- 页面跳转和传参



### 学习建议

1）ArkUI 采用声明式 UI，和传统的命令式 UI（如 Android XML）完全不同。要转变思维方式，理解声明式编程的理念。

![](https://pic.yupi.icu/1/image-20251127145536078.png)

2）装饰器是 ArkUI 的核心，要理解各种装饰器的作用（@Entry、@Component、@State、@Prop 等）。

3）DevEco Studio 是鸿蒙开发的官方 IDE，功能强大，要熟练掌握其使用。

4）这个阶段要多动手，尝试用 ArkUI 开发简单的界面，熟悉组件和布局的使用。



### 学习资源

- ⭐ [HarmonyOS 官方文档](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides-V5/application-dev-guide-V5)：官方文档
- [黑马程序员鸿蒙 5.0 教程](https://www.bilibili.com/video/BV1gSZvYzEdZ)：零基础到实战
- [鸿蒙开发入门到实战](https://developer.huawei.com/consumer/cn/forum/topic/0202177355322792038)：官方教程



## 阶段 3：鸿蒙特色能力（15-20 天，仅供参考）


鸿蒙特色能力包括分布式能力、原子化服务、一次开发多端部署等，是鸿蒙区别于其他平台的核心优势。



### 学习目标

掌握鸿蒙的特色能力，如分布式能力、多设备协同等。



### 知识点

**分布式能力【必学】：**

- 分布式数据管理
- 分布式任务调度
- 分布式文件系统

**多设备协同【建议学】：**

- 多设备发现和连接
- 跨设备迁移
- 跨设备协同

**系统能力【必学】：**

- 网络请求（http）
- 数据存储（Preferences、关系型数据库）
- 通知（Notification）
- 后台任务

**AI 能力【建议学】：**

- 调用盘古大模型 API
- AI 语音识别
- AI 图像识别



### 学习建议

1）分布式能力是鸿蒙的核心特色，区别于 Android 和 iOS。要理解分布式的概念和应用场景。

2）鸿蒙的 AI 能力非常强大，内置了华为的 AI 服务。可以方便地调用 AI 接口，为应用增加智能功能。

3）这部分内容相对高级，建议先掌握前面的基础知识，再学习鸿蒙特色能力。



### 学习资源

- [HarmonyOS 分布式能力 - 同应用跨设备数据同步](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/distributed-data-sync)：官方文档



## 阶段 4：项目实战（20-45 天，仅供参考）

### 学习目标

通过实际项目巩固所学知识，积累项目经验。



### 学习建议

1）从简单项目开始：先开发一些简单的应用，如待办事项、天气查询、笔记应用等。

2）参考官方示例：华为提供了大量示例项目，可以学习和参考。

3）开发原生鸿蒙应用：2024 年后，华为要求所有应用都要开发鸿蒙原生版本。可以将现有的 Android 应用改造成鸿蒙应用。

4）发布应用：完成项目后，可以发布到华为应用市场，体验完整的应用发布流程。

5）利用鸿蒙特色：在项目中应用鸿蒙的特色能力（如分布式、多设备协同），增加项目的亮点。



### 项目推荐

**入门级项目：**

- 待办事项
