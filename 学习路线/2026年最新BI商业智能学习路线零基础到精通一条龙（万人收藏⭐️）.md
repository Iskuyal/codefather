## 开篇介绍

BI（Business Intelligence，商业智能）是指将企业的数据转化为有价值的商业洞察的技术和方法。BI 通过数据收集、数据处理、数据分析、数据可视化等手段，帮助企业管理者做出更明智的决策。BI 系统一般包括数据仓库、ETL 工具、OLAP 分析、报表系统、Dashboard 等组件。

BI 的发展已经有 30 多年历史，从早期的报表系统到现代的自助式 BI，BI 技术在不断演进。传统 BI 需要 IT 人员开发报表，业务人员只能查看；现代自助式 BI 让业务人员可以自己拖拽分析，大大提高了数据分析的效率。2025 年，**AI 驱动的 BI 成为新趋势**，通过自然语言问答、智能推荐等功能，让数据分析更加智能化。

![](https://pic.yupi.icu/1/power-bi-artificial-intelligence-Main.jpg)

**为什么要学 BI？**

BI 是企业数据分析的核心工具，几乎所有企业都在使用 BI 系统。BI 工程师负责 BI 系统的开发和维护，数据分析师使用 BI 工具进行数据分析。掌握 BI 技术，不仅能让你开发 BI 系统，还能提升数据分析能力。BI 相关岗位的薪资不错，一线城市的 BI 工程师平均薪资在 18-35K。

BI 的主流工具包括：Tableau（可视化能力强）、Power BI（微软生态整合好）、FineBI（国产 BI 工具）、QlikView 等。Tableau 和 Power BI 是国际主流，FineBI 是国内市场占有率第一。这些工具各有特点，要根据企业需求选择合适的工具。

![](https://pic.yupi.icu/1/Power-BI-las-ultimas-actualizaciones-que-impulsan-el-analisis-de-datos.png)

要注意，BI 不等于数据分析！数据分析是方法论，BI 是工具。数据分析师使用 BI 工具进行数据分析，BI 工程师开发和维护 BI 系统。两者相辅相成。

## 整体学习建议

1）先学数据分析：BI 是数据分析的工具，要先理解数据分析的方法和流程，再学习 BI 工具。

2）选择一个工具：Tableau、Power BI、FineBI 选一个深入学习即可。建议选择 Power BI（免费且功能强大）或 FineBI（国产且本土化好）。

不过很多 BI 都是商业级付费产品，如果你目标不是成为专业的 BI 工程师或者数据分析师，可以直接看鱼皮的 [这期视频](https://www.bilibili.com/video/BV1CkUDBiEMR)，来了解主流的、免费的网站分析 / 数据分析工具，说不定能帮你节省很多钱~

4）BI 的核心是数据可视化，要学习各种图表类型（柱状图、折线图、饼图、散点图等）的使用场景和设计原则。

5）学习 BI 时可以用 AI 工具辅助理解概念、分析数据。可以到鱼皮的 [AI 资源大全](https://ai.codefather.cn/) 中找找工具。

## 阶段 1：BI 基础（7-15 天，仅供参考）

### 学习目标

理解 BI 的基本概念和架构。

### 知识点

**BI 基础概念【必学】：**

- BI 的定义和价值
- BI 的组成（数据仓库、ETL、OLAP、报表）
- BI 的应用场景

**数据可视化基础【必学】：**

- 图表类型（柱状图、折线图、饼图、散点图等）
- 图表选择原则
- Dashboard 设计

**BI 工具对比【建议学】：**

- Tableau vs Power BI vs FineBI
- 工具选型



### 学习重点

1）BI 的价值是将数据转化为洞察，帮助企业做决策。要理解 BI 在企业中的作用和应用场景。

2）不同图表适合不同的数据和分析目的。柱状图适合对比，折线图适合趋势，饼图适合占比，散点图适合相关性分析。

## 阶段 2：Power BI / FineBI（20-30 天，仅供参考）


Power BI 和 FineBI 是目前最流行的商业智能工具，能够快速实现数据可视化和报表制作，是 BI 开发的核心技能。



### 学习目标

掌握一个 BI 工具，能够开发报表和 Dashboard。



### 知识点

**Power BI 基础【建议学】：**

- Power BI 的安装和使用
- 数据导入和建模
- DAX 函数
- 报表设计

**FineBI 基础【建议学】：**

- FineBI 的使用
- 自助分析
- 组件开发

**Tableau 基础【建议学】：**

- Tableau 的使用
- 可视化开发



### 学习建议

1）Tableau、Power BI、FineBI 是三大主流 BI 工具。建议优先学习 Power BI 或 FineBI。Power BI 免费且功能强大，FineBI 国产且本土化好。

2）DAX 是 Power BI 的公式语言，类似于 Excel 的公式，用于数据建模和计算。要掌握常用的 DAX 函数。

![](https://pic.yupi.icu/1/2020-04-16_11h32_57.png)

3）这个阶段要多动手，尝试用 BI 工具开发各种报表和 Dashboard，熟悉工具的使用。
