## 开篇介绍

C#（读作 C Sharp）是微软开发的一门现代面向对象编程语言，于 2000 年随 .NET 框架一起发布。

C# 结合了 C++ 的强大功能和 Visual Basic 的易用性，语法简洁优雅，功能强大，是微软生态的核心编程语言。

虽然 C# 最初是为了开发 Windows 桌面应用而设计的，但随着 .NET 框架的发展，C# 的应用范围越来越广。现在 C# 不仅可以开发 Windows 桌面应用，还可以开发移动应用（Xamarin、MAUI）、游戏（Unity）、Web 应用（ASP.NET）、云服务等。

**为什么要学 C#？**

C# 是一门非常全面的编程语言，掌握 C# 可以让你在多个领域工作。如果你想开发 Unity 游戏，C# 是必学的脚本语言；如果你想开发 Windows 桌面应用，C# 是首选语言；如果你想开发企业级 Web 应用，ASP.NET Core 也是一个选择。而且 C# 的就业面很广，从游戏公司到传统企业，从互联网公司到金融机构，都有 C# 的岗位需求。

在 AI 时代，C# 也可以用于 AI 应用开发（虽然没 Python 和 Java 的生态那么好）。微软推出了 ML.NET 机器学习框架、Semantic Kernel AI 编排框架，让 C# 程序员也能开发 AI 应用。

## 整体学习建议

1）C# 的语法设计得非常好，简洁优雅，比 Java 更现代、比 C++ 更简单。如果你有 Java 或 C++ 基础，学习 C# 会非常快。

2）关注 .NET 生态：C# 离不开 .NET，要了解 .NET 的核心库和框架，建议优先学习 .NET 8。

3）选择发展方向：C# 的应用范围很广，建议根据自己的兴趣选择方向（游戏开发、Web 开发、桌面应用等），重点学习相关的框架和技术。

4）多看微软官方文档：微软的官方文档写得非常好，遇到问题时，建议优先搭配 AI 工具来查阅微软官方文档。

## 阶段 1：C# 基础语法（10-20 天，仅供参考）

### 学习目标

掌握 C# 的基础语法，能够编写简单的 C# 程序。



### 知识点

**基础语法【必学】：**

- 变量和常量
- 数据类型（int、float、double、string、bool）
- 运算符和表达式
- 控制流（if、switch、for、while、do-while、foreach）
- 数组和集合

**方法【必学】：**

- 方法的定义和调用
- 参数和返回值
- 方法重载
- ref 和 out 参数

**字符串【必学】：**

- 字符串的创建和操作
- 字符串插值（$""）
- StringBuilder

**异常处理【必学】：**

- try-catch-finally
- throw
- 自定义异常



### 学习建议

1）C# 的开发工具推荐 Visual Studio（Windows）或 Visual Studio Code（跨平台）+ .NET SDK。Visual Studio 是最强大的 C# IDE，功能完善。

2）C# 的字符串插值非常方便，可以在字符串中嵌入表达式，比 string.Format() 更简洁。

3）foreach 是 C# 遍历集合的标准方式，比 for 循环更简洁。

4）这个阶段要多写代码，熟悉 C# 的基本语法。可以在 Visual Studio 中创建控制台应用练习。



### 学习资源

- ⭐ [C# 官方文档](https://learn.microsoft.com/zh-cn/dotnet/csharp/)：微软官方文档



## 阶段 2：面向对象编程（10-20 天，仅供参考）

### 学习目标

掌握 C# 的面向对象编程，理解类、对象、继承、多态等概念。



### 知识点

**类和对象【必学】：**

- 类的定义
- 字段、属性、方法
- 构造函数和析构函数
- this 和 base 关键字
- static 静态成员

**封装、继承、多态【必学】：**

- 访问修饰符（public、private、protected、internal）
- 继承
- 虚方法和重写（virtual、override）
- 抽象类和抽象方法（abstract）
- 接口（interface）
- 多态

**属性和索引器【必学】：**

- 自动属性
- 只读属性
- 索引器

**委托和事件【建议学】：**

- 委托（Delegate）
- 事件（Event）
- Lambda 表达式



### 学习重点

1）C# 的属性（Property）比 Java 的 getter/setter 更简洁，可以直接定义属性而不需要写方法。

2）接口在 C# 中非常重要，C# 不支持多继承，但支持实现多个接口。

3）委托和事件是 C# 的特色功能，可以实现回调和观察者模式。虽然相对复杂，但在 Unity 和 ASP.NET 中非常常用。

4）Lambda 表达式可以让代码更简洁，特别是在 LINQ 和事件处理中。



### 学习资源

- [C# 面向对象编程](https://learn.microsoft.com/zh-cn/dotnet/csharp/fundamentals/object-oriented/)：官方文档



## 阶段 3：C# 高级特性（15-20 天，仅供参考）

### 学习目标

掌握 C# 的高级特性，比如泛型、LINQ、异步编程等，让代码更加优雅和高效。



### 知识点

**泛型【必学】：**

- 泛型类和泛型方法
- 泛型约束
- 泛型集合（List<T>、Dictionary<TKey, TValue>）

**LINQ【必学】：**

- LINQ 查询语法
- LINQ 方法语法
- 常用 LINQ 操作（Where、Select、OrderBy、GroupBy）

**异步编程【必学】：**

- async 和 await
- Task 和 Task<T>
- 异步方法的编写

**扩展方法【建议学】：**

- 扩展方法的定义和使用

**空值处理【必学】：**

- 可空类型（Nullable<T> 和 ?）
- 空值合并运算符（??）
- 空条件运算符（?.）

**新特性（C# 9-12）【建议学】：**

- 记录类型（record）
- 模式匹配
- init 访问器
- 主构造函数（C# 12）
- 集合表达式（C# 12）



### 学习重点

1）LINQ 是 C# 的强大特性，可以用统一的语法查询各种数据源（数组、集合、数据库、XML 等）。LINQ 在数据处理中非常常用，一定要掌握。

2）async/await 是 C# 异步编程的标准方式，可以让异步代码看起来像同步代码，大大提高了代码的可读性。

3）C# 每年都会发布新版本，引入新特性。建议多关注 C# 的最新特性，保持技术更新。

4）这个阶段的内容相对高级，建议先确保掌握了前面的基础知识，再学习高级特性。



### 学习资源

- [C# LINQ 教程](https://learn.microsoft.com/zh-cn/dotnet/csharp/linq/)：官方文档
- [C# async/await 教程](https://learn.microsoft.com/zh-cn/dotnet/csharp/asynchronous-programming/)：官方文档



## 阶段 4：.NET 框架和应用开发（15-30 天，仅供参考）


.NET 框架提供了丰富的类库和工具，支持 Web、桌面、移动等多种应用开发，是 C# 开发的重要基础。



### 学习目标

掌握 .NET 的核心框架，能够开发实际的应用。

💡 下面的知识点要根据自己感兴趣的方向进行选择。



### 知识点

**ASP.NET Core（Web 开发）：**

- ASP.NET Core 的安装和使用
- MVC 模式
- 路由和控制器
- Razor 页面

- Entity Framework Core（ORM）

**WPF（桌面应用）：**

- WPF 的基础概念

- Unity 中的 C# 脚本
- MonoBehaviour 生命周期
