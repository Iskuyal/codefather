## 行前须知

### 🌤️ 七月天气

| 项目 | 详情 |
|------|------|
| 平均气温 | 28°C – 36°C |
| 天气特征 | 高温湿热、午后偶有雷阵雨 |
| 日出时间 | 约 5:30 – 5:45 |
| 日落时间 | 约 19:00 – 19:10 |
| 台风可能 | ⚠️ 7月为台风高发期，出行前务必关注天气预报 |

### 🚗 自驾路线（深圳 → 潮州）

| 项目 | 详情 |
|------|------|
| 推荐路线 | 水官高速 → **沈海高速 (G15)** → 潮惠高速 → 汕昆高速 |
| 总里程 | 约 **350 – 360 公里** |
| 耗时 | 约 **3.5 – 4.5 小时**（视路况） |
| 高速过路费 | 约 **¥170 – 190**（单程，小型客车） |
| 油费 | 约 **¥220 – 280**（单程，1.6L排量参考） |
| 单程总交通费 | 约 **¥390 – 470** |

> 💡 **出发建议**：早上 **7:00 前**从深圳出发，避开早高峰，中午前即可抵达潮州。

### 🅿️ 古城停车指南

- **北关停车场**（官方推荐）：车位充裕，2026年古城交通管制升级，外地车建议停此
- **人民广场地下停车场**：步行至牌坊街约15分钟
- ⚠️ **古城核心区严禁驾车进入**，民宿多在巷弄内，车辆无法直达，需提前联系民宿老板确认最近停车点及接驳方式

### 🎒 七月必备物品清单

| 类别 | 物品 | 备注 |
|------|------|------|
| 🧴 防晒 | 防晒霜（SPF50+）、遮阳帽、墨镜、冰袖 | 7月紫外线极强 |
| 🌧️ 防雨 | 折叠伞、轻便雨衣 | 午后可能突降雷阵雨 |
| 🦟 防蚊 | 驱蚊液/驱蚊贴、止痒膏 | 古城水边蚊虫多，凤凰山更甚 |
| 👟 穿着 | 透气运动鞋/平底鞋、速干T恤、薄长裤 | 古城石板路走一天脚会累 |
| 🧥 保暖 | 轻薄外套/防晒衣 | 凤凰山顶夜晚降温至15-18°C |
| 📱 电子 | 充电宝（2万毫安以上）、车载手机支架 | 导航+拍照耗电快 |
| 💊 药品 | 藿香正气水、肠胃药、创可贴 | 防中暑+吃撑备用 |
| 🧻 日用 | 湿纸巾、挂脖小风扇 | 户外排队续命神器 |
| 🚗 车用 | 遮阳挡、胎压表 | 夏季高速前检查胎压（冷胎2.3bar） |

### 🍽️ 饮食禁忌重申

> ⚠️ **同行人员不食用生鲜、刺身类食物**（包括潮汕生腌、鱼生等）。本攻略所有餐饮推荐已自动排除生腌、鱼生类菜品，但不排除部分餐馆菜单中有此类选项，点餐时请注意避开。

## 避坑提醒

> 15年潮汕导游经验结晶

### 🚫 坑1：古城开车进去
> 古城内部道路2-3米宽，外地车进去进退两难。**正确**：车停北关停车场或人民广场，步行/共享电动车（¥3/次）逛古城。

### 🚫 坑2：周末/节假日上南澳岛
> 南澳大桥堵3-5小时是常事。**2026年限流1万辆/天**。**正确**：选周二至周四上岛，早上8:30前上桥。周末出行建议前一晚住汕头，早上6:30出发。

### 🚫 坑3：网红店盲目排队
> 杏花吴记排4小时不夸张。**正确**：提前公众号取号，或错峰14:00-16:00。备选**福合埕**（60年老店）、**八合里海记总店**同品质不等位。

### 🚫 坑4：凤凰山自驾上山
> 乌岽顶单车道+发卡弯，会车空间极窄。**正确**：导航"乃兴旅游集散中心"，换乘景区大巴。

### 🚫 坑5："时价海鲜"不确认
> 结账时价格可能远超预期。**正确**：逐道确认单价和重量，或去云澳渔港自选+加工（明码标价）。

### 🚫 坑6：7月不带外套上凤凰山
> 山下35°C → 山顶晚上15-18°C，温差近20度！**正确**：车上常备轻薄外套。

### 🚫 坑7：牌坊街主街"景区化"小吃
> 价格虚高、口味平庸。**正确**：拐进**西马路、南春路、义安路**，价格便宜一半，味道正宗三倍。

### 🚫 坑8（新增）：周一踩空
> 广济桥**周一白天闭桥维护**，**周一晚无灯光秀**！韩文公祠也周一闭馆。**正确**：周一安排凤凰山、龙湖古寨等不受影响的行程。

### 🚫 特别提醒：夏季台风
> 7月粤东台风高发。出行前一周关注天气预报，若台风正面影响，南澳大桥可能关闭。
