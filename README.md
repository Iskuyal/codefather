## ⚡ 快速参考

### 🚗 深圳 → 潮州自驾（2辆车）
- 路线：沈海高速 (G15)，约 350-360 公里
- 耗时：约 3.5-4.5 小时
- 单程总交通费：约 ¥860-940（2车合计）

### ⚠️ 重要提醒
- **周一**广济桥闭桥 + 无灯光秀，韩文公祠闭馆
- 南澳岛建议**周二至周四**上岛，8:30 前上桥
- 凤凰山私家车禁上山，导航「乃兴旅游集散中心」换乘大巴
- 7月为台风高发期，出行前关注天气
- **大团队出行**：建议提前建微信群，共享实时位置
