## ⚡ 快速参考

### 🚗 深圳 → 潮州（2辆车）
- 路线：沈海高速 (G15)，约 350-360 公里
- 耗时：约 3.5-4.5 小时
- 过路费约 ¥170-190/车（单程）

### ⚠️ 关键提醒
- **周一**：广济桥闭桥 + 无灯光秀，韩文公祠闭馆
- **南澳岛**：建议周二至周四上岛，8:30前上桥
- **凤凰山**：私家车禁上山，导航「乃兴旅游集散中心」
- **大团队**：提前建微信群共享位置，餐厅务必提前订桌
